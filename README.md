TasksHub is a full-stack web application designed for personal productivity. It features a dual-purpose interface that allows users to manage granular tasks while maintaining a repository for long-form documentation.

Authentication: Secure sign-up/login flow to protect user data.

Task Engine: CRUD functionality for task tracking and status management.

Docs Module: A dedicated markdown-friendly (or rich-text) area for persistent documentation.

All-in-One UI: A clean, centralized dashboard for maximum efficiency.
